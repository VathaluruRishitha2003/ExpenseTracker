# ExpenseTracker
Python app to track personal expenses, categorize spending, view summaries, and generate charts for easy financial management.
# Expense Tracker

## Description
A Python project to track personal expenses with categories, summaries, and visualizations.

## Features
- Add, edit, delete expenses
- View expense summaries
- Generate charts for spending

## Installation
1. Clone the repo
2. Install dependencies using `pip install -r requirements.txt`
3. Run `python main.py`

## Usage
Follow the prompts to add your expenses.
